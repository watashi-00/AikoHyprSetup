# AikoHyprSetup - Widget Roadmap & Ideas

This document keeps track of brainstorming ideas and widgets for the AikoHyprSetup environment.

## 🟩 Implemented Features & Widgets

- [x] **Aiko-Search** (Implemented in v2): Spotlight-style search panel and app launcher with autocompletes, YouTube search, and active highlights.
- [x] **Aiko-Calendar** (Implemented in v2): Interactive, theme-aware desktop calendar widget.
- [x] **Aiko-Timer** (Implemented in v2): Countdown and Pomodoro productivity timer.
- [x] **Aiko-Bluetooth** (Implemented in v3): Dedicated interactive GTK Bluetooth manager for device discovery and connection.
- [x] **Aiko-Recorder** (Implemented in v2): Screen and audio recording dashboard wrapping `wf-recorder`.
- [x] **Aiko-LiveWallpaper** (Implemented in v3): Dynamic live/video wallpaper selector and synchronization manager utilizing `mpvpaper`.

*Feel free to brainstorm and append new ideas here as the project evolves.*
