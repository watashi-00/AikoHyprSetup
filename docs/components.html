Waybar Components

All Waybar elements can be styled using their CSS IDs or classes in waybar/style.css.

#custom-spotify - Music Info

Displays the currently playing song and artist.

#custom-spotify {
    min-width: 160px;
    color: #ffc2dc;
}

Hyprland Theme Tags

Special comments used by the theme-selector.sh script to update Hyprland settings dynamically.

Theme Markers

Add these comments at the end of the line in configs/hypr/hyprland.conf to make them theme-able:

  • # @theme:gaps_in: Inner gaps between windows.
  • # @theme:gaps_out: Outer gaps (screen edge).
  • # @theme:active_border: Gradient for active windows.
  • # @theme:rounding: Corner radius.
  • # @theme:active_opacity: Transparency for active window.

Custom Scripts

Located in the scripts/ directory.

  • wallpaper.sh: Graphical wallpaper selector with cropping.
  • theme-selector.sh: Dynamic theme engine switcher.
  • power-menu.sh: System power options via Wofi.
