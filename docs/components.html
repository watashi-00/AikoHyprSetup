Component & CSS Tag Documentation

Waybar Components

All Waybar elements can be styled using their CSS IDs or classes in themes/*.css (which are linked to waybar/style.css).

#custom-spotify - Music Info

Displays the currently playing song and artist.

#custom-spotify {
    min-width: 160px;
    color: #ffc2dc;
}

#spotify-buttons - Media Controls

A container group for the media control buttons.

  • #custom-spotify-prev: Previous track button.
  • #custom-spotify-play: Play/Pause toggle button.
  • #custom-spotify-next: Next track button.

#workspaces - Workspace Switcher

The 1-5 navigation module.

#workspaces button.active {
    background-color: #ff8fbd;
    color: #261621;
}

#taskbar - Dynamic Taskbar

The taskbar with themed icons. Uses the Aiko icon theme generated dynamically.

#taskbar button image {
    /* Filters are not supported in Waybar, use icon-gen system */
}

Aiko Widgets

Standalone Python/GTK 3 widgets that respond to the global theme system via theme.css symlink.

Aiko Note

Simple sticky note widget. Linked via @widget-aiko-note tag.

  • ID: #aiko-note-window
  • Content: #note-content

Aiko Player

Rich media player widget with album art. Linked via @widget-aiko-player tag.

  • ID: #aiko-player-window
  • Controls: #player-btn-main, #player-btn-small
  • Progress: scale#player-progress

Aiko User Card

Personalized user info card. Linked via @widget-aiko-usercard tag.

  • ID: #aiko-usercard-window
  • Avatar: #usercard-avatar-container (circular clipping)

Global Theme Tags

Place these markers at the end of lines in configuration files. The theme-selector.sh script will search and replace values based on the theme header.

Configuration Markers

Hyprland (configs/hypr/hyprland.conf)

  • @theme:gaps_in: Inner gaps.
  • @theme:gaps_out: Outer gaps.
  • @theme:active_border: Active window border color/gradient.
  • @theme:inactive_border: Inactive window border color.
  • @theme:rounding: Corner rounding radius.
  • @theme:active_opacity: Active window opacity.
  • @theme:inactive_opacity: Inactive window opacity.

Wofi (configs/wofi/style.css)

  • @theme:wofi-bg: Window background.
  • @theme:wofi-border: Window border.
  • @theme:wofi-text: Text color.
  • @theme:wofi-accent: Selection color.

Mako (configs/mako/config)

  • @theme:mako_bg: Background color.
  • @theme:mako_text: Text color.
  • @theme:mako_border: Border color.
  • @theme:mako_rounding: Corner radius.

Waybar (waybar/config.jsonc)

  • @theme:waybar_accent: Primary color for icons and interactive elements (Pango span colors).

Dynamic Widget Mapping

Define these in the theme CSS header (e.g., themes/pink-anime.css):

  • @widget-aiko-note: CSS file for Note widget.
  • @widget-aiko-player: CSS file for Player widget.
  • @widget-aiko-clock: CSS file for Clock widget.
  • @widget-aiko-usercard: CSS file for UserCard widget.
  • @widget-aiko-weather: CSS file for Weather widget.
  • @widget-aiko-list: CSS file for List widget.
  • @widget-aiko-sys: CSS file for System widget.
  • @widget-aiko-calendar: CSS file for Calendar widget.
  • @widget-aiko-timer: CSS file for Timer widget.
  • @widget-aiko-recorder: CSS file for Recorder widget.
  • @widget-aiko-search: CSS file for Search widget.
