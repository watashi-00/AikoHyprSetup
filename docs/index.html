Aesthetic desktop. Zero effort.

A versatile, highly customizable environment for Hyprland and Waybar. Equipped with elegant dynamic wallpaper color syncing and powerful modular desktop widgets.

Get Started Release Notes
watashi@arch ~ AikoHyprSetup
Powerful Engine

Redesigned in v3

Experience fluid animations, dynamic wallpaper syncing, and deep hardware optimization directly out of the box.

Dynamic Themes & Wallpapers

Automatically extract primary color palettes from static or video wallpapers, dynamically syncing accents across your entire system.

Interactive Bluetooth Manager

Scan, pair, connect, and configure Bluetooth devices natively using our completely redesigned GTK Bluetooth widget.

GPU Tuning (Nvidia & Multi-GPU)

Optimize rendering instantly. Automate WLR/DRM settings for dual-GPU/Nvidia layouts using the built-in configuration helper.

Smart Fullscreen Automation

Event-listener daemons automatically pause resource-heavy wallpapers (mpvpaper) and audio visualizers when applications go fullscreen.

Modular GTK Widgets

A complete floating dashboard containing Media Controllers, System Resource Monitors, oversized Clocks, Notes, and volume mixers.

Universal Installer

Robust installers with automated backups and dependency managers for Arch (with yay/paru helper support), Fedora, Debian, and Alpine.

Get Started

Install in Seconds

Run the universal installer command on your terminal. The script will safely create backups of existing files automatically.

$
git clone https://github.com/watashi-00/AikoHyprSetup.git && cd AikoHyprSetup && ./install.sh
