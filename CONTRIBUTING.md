# Contributing to AikoHyprSetup

First off, thank you for considering contributing to AikoHyprSetup! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
*   **Check the FAQ**: Before reporting a bug, check if it's already a known issue.
*   **Use the Template**: When opening an issue, use the bug report template and provide as much detail as possible.

### Suggesting Enhancements
*   **Be Specific**: Explain why the enhancement would be useful and how it should work.
*   **Search First**: Check if someone has already suggested something similar.

### Pull Requests
1.  Fork the repository and create your branch from `master`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the test suite passes.
4.  Make sure your code follows the existing style (naming conventions, formatting).
5.  Write a clear, concise commit message.

## Style Guide
*   **Scripts**: Use ShellCheck to validate your bash scripts.
*   **Python**: Follow PEP 8 guidelines for widgets.
*   **CSS**: Keep themes modular and use the `@theme:` markers for dynamic patching.

## Community
*   Join the discussion in the Issues or Pull Requests.
*   Be respectful and follow our [Code of Conduct](CODE_OF_CONDUCT.md).

Happy hacking!
