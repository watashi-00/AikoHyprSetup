# Security Policy

## Supported Versions

Only the latest version of AikoHyprSetup is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within AikoHyprSetup, please do not open a public issue. Instead, report it privately to the maintainer.

### Where to report?
Please send an email to @watashi-00/nova2948@gmail.com. or open a private security advisory on GitHub if available.

### What to include?
*   A detailed description of the vulnerability.
*   Steps to reproduce the issue.
*   The potential impact of the vulnerability.
*   (Optional) A proposed fix or mitigation.

We appreciate your help in keeping AikoHyprSetup secure!
